# Credit Card Fraud Detection System

## Project Description

This project predicts whether a credit card transaction is Legitimate or Fraudulent using Machine Learning.

## Technology Used

- Python
- Flask
- Random Forest Classifier
- Scikit-Learn
- Bootstrap
- HTML
- CSS

## Project Structure

```
app.py
credit_card_model.pkl
requirements.txt
templates/
static/
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```

## Model

Random Forest Classifier
